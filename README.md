# Belajar Python dari Nol

Repository ini berisi materi belajar Python untuk pemula yang ingin memahami coding dari dasar, bukan sekadar menghafal syntax.

Materi ini dibuat dengan prinsip sederhana:

> Kalau sebuah konsep bisa dijelaskan lebih jelas, maka jangan dibuat rumit.

## Tujuan

- Menjelaskan Python dengan bahasa yang mudah dipahami.
- Mengurangi kebingungan pemula saat belajar coding.
- Menjawab hal-hal kecil yang sering tidak dijelaskan di sekolah.
- Membantu saya belajar Python dengan cara menyusun dan menjelaskan ulang materi.

## Cara Belajar di Repository Ini

Setiap materi akan dibuat dengan pola:

1. Penjelasan singkat
2. Contoh kode
3. Cara membaca kode
4. Kesalahan umum pemula
5. Latihan sederhana

## Materi

- Apa itu Python
- Variabel
- Input dan Output
- Tipe Data
- Operator
- Percabangan
- Perulangan
- Function
- Proyek Sederhana

## Untuk Siapa?

Repository ini cocok untuk:

- Siswa SMK RPL
- Pemula yang baru belajar coding
- Orang yang sering bingung karena penjelasan terlalu teknis
- Siapa pun yang ingin belajar Python secara pelan, jelas, dan tidak ambigu

## Catatan

Saya juga masih belajar. Repository ini bukan dibuat karena saya sudah paling ahli, tetapi karena proses menyusun materi adalah bagian dari cara saya memahami Python.

Kesalahan akan diperbaiki, materi akan berkembang, dan pemahaman akan bertambah seiring waktu.
